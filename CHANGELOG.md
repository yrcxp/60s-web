# Changelog

## 0.1.1

- 增加移动端导航模式设置，支持自动、底部导航和顶部导航。
- 新安装用户默认自动模式：PWA 独立窗口使用底部导航，普通浏览器使用顶部导航。
- 已有本地设置的 0.1.0 用户默认保留底部导航，避免升级后突然改变操作习惯。
- 配置导入导出和恢复默认同步支持移动端导航模式。
- 修复 iOS PWA 底部导航栏安全区空白，并在下滑阅读时自动隐藏底部导航。
- iOS 添加到主屏幕提示关闭后会记住选择，刷新后不再重复显示。
- 更新 Service Worker 静态缓存名到 `60s-web-static-v0.1.1`。

## 0.1.0

首个面向公开使用的稳定整理版。

- 完成首页、热榜、新闻、天气、工具和设置页的核心体验收口。
- 增加配置导入导出、API 连接检测、非法 API 地址容错和恢复默认设置。
- 增加接口实验室常用接口收藏，并支持配置导入导出。
- 增加首页快捷入口收藏，并支持设置页管理、配置导入导出和恢复默认。
- 增强 iOS PWA 支持，加入安装提示、离线提示、更新提示和移动端底部导航。
- 更新 Service Worker 静态缓存名到 `60s-web-static-v0.1.0`。
- 清理未引用的 `favicon.svg` 静态资源。
